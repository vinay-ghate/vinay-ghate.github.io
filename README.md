# Vinay Ghate - Portfolio Website

A modern, responsive personal portfolio website showcasing my work as a System Engineer & AI Developer.

## About

This portfolio website features:
- **Dynamic Content**: All data loaded from JSON for easy updates
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Modern UI**: Clean, professional design with smooth animations
- **Fast Loading**: Optimized for quick page loads

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: Ionicons
- **Fonts**: Google Fonts (Poppins)
- **Data Management**: JSON-based content system

## Features

- ✅ Personal information and contact details
- ✅ Education and professional experience
- ✅ Project portfolio with filtering
- ✅ Technical skills showcase
- ✅ Activities and achievements
- ✅ Coding profiles and social links
- ✅ Blog section
- ✅ Contact form with direct email integration

## Quick Start

1. Clone or download the repository
2. Open `index.html` in your browser
3. To update content, edit `data.json` or modify the embedded data in `assets/js/data-loader.js`

## Project Structure

```
├── index.html              # Main portfolio page
├── data.json              # Portfolio data (for reference)
├── assets/
│   ├── js/
│   │   ├── data-loader.js # Dynamic content loader
│   │   └── script.js      # Interactive functionality
│   ├── css/
│   │   └── style.css      # Styling
│   └── images/            # Portfolio images
└── README.md              # This file
```
